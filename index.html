QZ Tray Installer

Automated installer for QZ Tray with SSL certificate generation.

Quick Install

PowerShell (Windows/Linux/macOS)

irm https://aprilboiz.github.io/qz-installer/install.ps1 | iex

Bash (Linux/macOS)

curl -fsSL https://aprilboiz.github.io/qz-installer/install.sh | bash

Direct Downloads

  • install.ps1 - PowerShell installer
  • install.sh - Bash installer

Usage Options

Install specific version:

irm https://aprilboiz.github.io/qz-installer/install.ps1 | iex -ArgumentList "2.2.5"

Install beta version:

irm https://aprilboiz.github.io/qz-installer/install.ps1 | iex -ArgumentList "beta"

Show help:

irm https://aprilboiz.github.io/qz-installer/install.ps1 | iex -ArgumentList "help"

What It Does

  1. Downloads the latest QZ Tray installer for your platform
  2. Detects and stops QZ Tray if running
  3. Installs QZ Tray silently
  4. Generates SSL certificates for localhost + your primary IPv4
  5. Restarts QZ Tray automatically
